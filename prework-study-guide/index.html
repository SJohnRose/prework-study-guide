HTML

  • The head element contains information about the web page
  • The body element represents the visible content shown to the user
  • The img element is used to add images and alt attribute is important
  • Some tags are semantic while others like div are non-semantic

CSS

  • CSS is used to add aesthetics to a web page
  • CSS can be inline, internal or external
  • CSS rule is made of selector{property: value;}
  • margin property indicates how much space we want outside an element
  • padding property adds space around the content inside an element
  • CSS uses the box model of HTML element with margin, border, padding and content

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git add -A: commits all files in the working branch
  • git commit -m "": commits changes to the repository with a short message
  • git pull origin main: pulls in the base branch into feature branch before pushing code
  • git push origin feature-name: pushes changes to base branch
  • git branch: shows the current active branch with an *

Javascript

  • JavaScript adds interactivity to websites
  • var is used to create a variable
  • console.log() is the output statement
  • for is used to perform actions repeatedly
  • if with else is used to check for conditions
  • function keyword is used to create functions
  • functions need to be defined and then called
  • an array is used to hold multiple values with an index
